# PandaPlug Vizsgaprojekt

## Struktura
- WEB (frontend + backend)
- MOBIL (frontend + backend)

## Futtatas

### Web backend
cd WEB/backend_PP
npm install
node app.js
npm start

### Web frontend
cd WEB/frontend_pp
npm install
npm start

### Mobil backend
cd MOBIL/backend_mobil
npm install
npm start

### Mobil frontend
cd MOBIL/frontend_pp
npm install
npx expo start
